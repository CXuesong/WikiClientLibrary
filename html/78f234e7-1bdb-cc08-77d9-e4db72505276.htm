Remarks

For common implementations for MediaWiki generator, prefer inheriting from WikiPagePropertyGenerator, which returns WikiPageStub for EnumItemsAsync implementation.

For generators, it's not necessary for the sequence taken out from EnumPagesAsync(IWikiPageQueryProvider) to be kept ordered. If you need ordered sequence, for example, enumerating the links from the beginning of an article using LinksGenerator, use EnumItemsAsync.
