The WikiPageExtensions type exposes the following members.

Methods
 NameDescription
CreateCategoriesGenerator Creates a CategoriesGenerator instance from the specified page, which enumerates the categories used on the page.
CreateFilesGenerator Creates a FilesGenerator instance from the specified page, which generates files from all used files on the page.
CreateLinksGenerator Creates a LinksGenerator instance from the specified page, which generates pages from all links on the page.
CreateRevisionsGenerator Creates a RevisionsGenerator instance from the specified page, which enumerates the sequence of revisions on the page.
CreateTransclusionsGenerator Creates a TransclusionsGenerator instance from the specified page, which generates pages from all pages (typically templates) transcluded in the page.
