WikiLink Properties

[This is preliminary documentation and is subject to change.]

The WikiLink type exposes the following members.

Properties
  NameDescription
Anchor
For wikilink expression in the form [[target|anchor]], excluding the square brackets, gets the actual displayed text (anchor) for the link.
DisplayText
Gets the actual link text that should be shown.
FullTitle
Title of the page, including namespace name, if exists.
InterwikiPrefix
Interwiki prefix of the wikilink. If this link is not an interwiki link, the value is null.
Namespace
The namespace information.
NamespaceName
Namespace name. If the title is in main namespace, the value is empty string. If this is not applicable (e.g. parsed a interwiki link without family information given), the value is null.
OriginalText
Gets the original wikitext expression that was passed to the Parse or ParseAsync methods.
Section
The section title of a section on the page, without leading #.
Site
The original wiki site provided to resolve this wikilink.
Target
For wikilink expression in the form [[target|anchor]], excluding the square brackets, gets the full page title and section title (target) for the link, including interwiki prefix.
TargetSite
Gets the wiki site containing the specified page title. If the parsed wikilink expression does not contain interwiki prefix, this property is the same as Site. If this wikilink is parsed with no IWikiFamily provided, while it contains interwiki prefix, this property will be null.
TargetUrl
Gets the full URL of the wikilink target.
Title
Title, excluding namespace and section.
