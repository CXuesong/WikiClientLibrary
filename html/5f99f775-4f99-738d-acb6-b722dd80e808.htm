Remarks
The implementation of this method involves
  • Generating HttpRequestMessage from request;
  • Transmitting HttpRequestMessage, and gets the HttpResponseMessage;
  • Parsing the HttpResponseMessage using responseParser (see ParseResponseAsync(HttpResponseMessage, WikiResponseParsingContext));
  • Retrying if possible;
  • Returning the parsed result, or throwing an exception.
