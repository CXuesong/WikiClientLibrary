Property Value

Type: Int32
Maximum count of items returned per MediaWiki API invocation. This limit is 10 by default, and can be set as high as 500 for regular users, or 5000 for users with the apihighlimits right (typically bots and sysops).
Remarks
This property decides how many items returned at most per MediaWiki API invocation.
