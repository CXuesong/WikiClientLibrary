Welcome to Wiki Client Library!

Wiki Client Library is a hand-crafted asynchronous MediaWiki API client library for wiki sites (including Wikipedia and its sister projects, as well as Wikia). This library aims for human users, as well as bots. This package is now available on NuGet. You may install the package using the following command in the Package Management Console

Copy
Install-Package CXuesong.MW.WikiClientLibrary -Pre
