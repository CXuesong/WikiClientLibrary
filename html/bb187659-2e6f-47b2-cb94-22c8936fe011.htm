WikiClientLibrary.Generators Namespace

[This is preliminary documentation and is subject to change.]

This namespace contains the implementations for various MediaWiki lists and generators.

Remarks

MediaWiki lists and generators allows you to enumerate through all the pages or other item sequence, with pagination support. In Wiki Client Library, these functionalities are implemented using IAsyncEnumerable from Ix.Async package.

For basic concepts on MediaWiki lists and generators, see mw:API:Lists and mw:API:Generator.
