site
Type: WikiClientLibrary.SitesWikiSite
The destination site to upload the file.
sourceStream
Type: System.IOStream
The seekable stream containing the file to be uploaded. The upload will be performed starting from the current position.
fileName
Type: SystemString
Optional file name. This parameter can be null, where a dummy file name will be used; otherwise, it should be a valid file name. If the name has file extension, the content will be validated by server. However, the name can be different from the actual uploaded file.
