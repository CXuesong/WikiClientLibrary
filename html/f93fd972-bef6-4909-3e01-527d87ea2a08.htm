Remarks
This exception represents the following MW API errors:
  • readapidenied – You need read permission to use this module.
  • permissiondenied
  • mustbeloggedin – You must be logged in to upload this file.
  • readapidenied
  • permissions
  • cantpurge
  • (empty ErrorCode) – Other cases where the user does not have the rights to perform the operation.
